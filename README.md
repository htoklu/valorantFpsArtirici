# valorantFpsArtirici
Valorant ve diğer oyunlar için geliştirilmiş, tek tıkla çalışan Windows performans optimize edici. Güç planı yönetimi, CPU önceliklendirme, DNS/Ağ temizliği, arka plan işlemlerini kısıtlama ve dizüstü bilgisayarlar için özel soğutma modları içerir. Tamamen güvenli ve kullanımı kolay profil yöneticisi desteği sunar.
